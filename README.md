# Company-Asset-Website
基于Spring Boot和Vue构建的公司资产管理网站，旨在实现高效的资产登记、查询与管理。后端使用Spring Boot提供强大的RESTful API，确保数据的安全与稳定；前端采用Vue框架，提供直观友好的用户界面。该系统支持多用户角色，权限管理，以及资产分类、报表生成等功能，帮助企业优化资产管理流程，提高运营效率。
